## 教程
建议**有一定搞机基础的人员**使用
1. 下载本仓库文件 并且一定要解压到纯英文目录里边   
2. 使用电脑连接手机并且**确保**你的手机**开启了ADB调试**并且模式改为传输文件   
3. 安装Python环境 [下载链接](https://www.123pan.com/s/chYrVv-eTRF3.html)   
4. 最后在文件夹中使用python脚本就行了
5. `python Replace.py`
